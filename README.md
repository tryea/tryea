## AI Engineering

Patterns built end to end and live:

- [FinRAG](https://finrag.ersaptaaristo.dev): RAG over real financial filings, with hybrid retrieval (pgvector + full-text, fused with reciprocal rank fusion), rerank, and grounded inline citations
- [AgentDesk](https://agentdesk.ersaptaaristo.dev): a support agent that investigates with tools, keeps memory across conversations, and pauses for human approval before any write
- [SQLGuard](https://sqlguard.ersaptaaristo.dev): plain English to SQL with three independent guardrail layers that block a destructive query before it reaches the database
- [SemCache](https://semcache.ersaptaaristo.dev): a semantic cache that returns a stored answer for a near-duplicate question, cutting model spend and latency
- [EvalForge](https://evalforge.ersaptaaristo.dev): turns production logs into a rubric test suite and scores a model with a separate LLM judge
- [LoRALab](https://loralab.ersaptaaristo.dev): a real LoRA fine-tune of a small model on a laptop GPU, with the before and after
